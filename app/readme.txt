se crea el layout en el que se importara el componente presentacional: navbar
se crea un componente para cada sesion del menu ingresar, ubicado en la barra de navegacion, estos componentes son: Loginusuario.jsx y Logineps.jsx
en cada componente se crean sus hojas de estilos 
los componentes Loginusuario y Logineps seran importados dentro del page.jsx correspondiente a cada uno respectivamente, esto es, el componente usuario se importaria dentro del page.jsx que esta ubicado dentro de la carpeta usuario, y el componente Logineps se importara dentro del page.jsx que esta ubicado dentro de la carpeta eps
en la carpeta de cuenta, se encuentra un page.jsx de prueba en e que se valida el ingreso con un mensaje por defecto para ambos clientes: usuarios o eps
